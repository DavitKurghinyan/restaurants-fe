# City Falcon FE

## Requirements
1. Node `>= v8.15.1`
2. NPM `>= v5`

## Setup Guide
Run `npm install`

## To run app in local machine
Run `npm run start`

## To build app
Run `npm run build`

It will create `build` folder with the already built code in the root of the app
